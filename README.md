Página web de la Delegación de Alumnos de la Escuela de Ingeniería Industrial de la Universidad de Vigo

## Instalación

### Linux (recomendado)

#### Utilizando `bun`

Para ejecutar la web en versión de desarrollo, utilizamos `bun`. Para 
instalarlo, sigue las instrucciones oficiales encontradas en
[la página oficial de bun](https://bun.sh/docs/installation)

Instala las dependencias ejecutando

```bash
bun install
```

Para iniciar el servidor de desarrollo ejecuta

```bash
bun dev
```

### Windows (probado en Windows 10)

Debido a que `bun` no está completamente implementado en Windows, utilizamos
`pnpm`. Para instalarlo, sigue las instrucciones encontradas en
[la página oficial de pnpm](https://pnpm.io/installation)

Instala las dependencias ejecutando

```powershell
pnpm install
```

Para iniciar el servidor de desarrollo ejecuta

```powershell
pnpm dev
```

Abre [http://localhost:3000](http://localhost:3000) en tu navegador (de preferencia Google Chrome)
para ver el resultado.
