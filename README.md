Página web de la Delegación de Alumnos de la Escuela de Ingeniería Industrial de la Universidad de Vigo

## Primeros pasos

### Instalar `bun`
Para ejecutar la web en versión de desarrollo, utilizamos `bun` en lugar de
`npm` por rapidez. Para instalarlo, hay que seguir las instrucciones oficiales
encontradas en [la página oficial de bun](https://bun.sh/docs/installation)

### Instalar las dependencias
Ejecuta el siguiente comando para instalar todas las dependencias que utiliza
la web
```bash
bun install
```

### Iniciar el servidor
Para iniciar el servidor de desarrollo ejecuta el siguiente comando.

```bash
bun dev
```

Abre [http://localhost:3000](http://localhost:3000) en tu navegador (de preferencia Google Chrome)
para ver el resultado.
